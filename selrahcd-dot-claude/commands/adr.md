# ADR - Architecture Decision Records

Manage Architecture Decision Records for documenting technical decisions.

## Usage
```
/adr new <title>
/adr list
/adr show <number>
```

Examples:
```
/adr new "Use PostgreSQL for persistence"
/adr list
/adr show 3
```

## Process

Invoke the `adr` skill to:
- Create new ADRs with proper structure and numbering
- List existing ADRs in the project
- Show details of a specific ADR
