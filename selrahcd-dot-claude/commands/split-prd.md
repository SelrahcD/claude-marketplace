# Split PRD - Transform PRD Features into User Stories

Transform technical PRD features into usable user stories.

## Usage
```
/split-prd <file>
```

Examples:
```
/split-prd docs/prd.md
/split-prd requirements/feature-spec.md
/split-prd SPEC.md
```

## Process

Invoke the `split-prd` skill to:
- Analyze PRD features and requirements
- Split features into well-defined user stories
- Apply INVEST criteria for story quality
- Create prioritized story backlog
